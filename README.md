## INSTALLATION
- Requirements: `Python`
```Powershell
# windows
python -m venv env
.\env\Scripts\activate
pip uninstall -r requirements.txt -y
    # quit of the virtual environment
    deactivate
```

## Run the app
```Powershell
# vanilla terminal
streamlit run app.py
# quit
ctrl-c
```
